# 🚀 Atividade Prática: Layout Dinâmico com Flexbox, Grid e Animações

Este projeto consiste na criação de um painel de controle responsivo, desenvolvido como parte da atividade prática de Web Design. O objetivo é aplicar conceitos de posicionamento estrutural e interatividade visual

## 📘 Contexto Técnico

De acordo com o roteiro da atividade, o projeto foca em dois pilares principais do CSS:

1.  **Transições e Animações**: Técnicas que criam efeitos visuais fluidos para melhorar a experiência do usuário
    **Transição**: Utilizada para suavizar mudanças de propriedades como cores e tamanhos
    ***Animação**: Utiliza `@keyframes` para definir movimentos complexos e estados iniciais/finais

## 🛠️ Requisitos Implementados

A aplicação foi construída seguindo rigorosamente os seguintes requisitos:

### 🧱 Estrutura e Layout
**Contêiner Principal**: Criado com três seções distintas: menu lateral, cabeçalho e conteúdo principal
**Grid Layout**: Utilizado para organizar o posicionamento global dessas três seções
Aplicado no **Menu Lateral** para alinhar os itens da lista verticalmente
Aplicado no **Cabeçalho** para distribuir o título e o botão

### 🎬 Efeitos e Interatividade
**Transição de Botão**: O botão no cabeçalho possui uma transição suave de cor ao passar o mouse (`hover`)
**Animação de Entrada**: As três caixas de conteúdo principal deslizam para dentro da página ao carregar, utilizando a regra `@keyframes slideIn`.
